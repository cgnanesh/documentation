# QikWork Terminology

## Glossary

- **Agile**: A methodology that promotes iterative development and collaboration.
- **Backlog**: A list of tasks or features that are yet to be completed.
- **Sprint**: A set period during which specific work has to be completed and made ready for review.
- **Epic**: A larger body of work that can be broken down into smaller tasks (user stories).
- **User Story**: A tool used in Agile development to capture a description of a feature from an end-user perspective.
- **Kanban**: A visual workflow management method that uses cards and columns to manage tasks.

Feel free to add more terms if needed!
